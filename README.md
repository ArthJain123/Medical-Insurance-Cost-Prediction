# 🏥 Medical Insurance Cost Prediction

## 📌 Project Overview

This project is created by me as a personal Machine Learning project to predict medical insurance charges based on various factors such as age, BMI, smoking status, number of children, region, and sex.

The project includes data cleaning, exploratory data analysis (EDA), feature encoding, and a Linear Regression model to understand the factors influencing insurance costs and predict future medical charges.

---

## 🎯 Objectives

* Perform data cleaning and preprocessing.
* Explore the dataset using visualizations.
* Identify factors affecting medical insurance charges.
* Build a Machine Learning model to predict insurance costs.
* Evaluate the model using regression metrics.
